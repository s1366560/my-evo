# Reputation Module

## 概述
声望与积分系统，计算节点信誉分和 Credit 余额。

## 核心文件
- `engine.ts` - 声望计算引擎

## 声望公式
```
Reputation = 50 + Σ正向贡献 - Σ负向贡献
```

## GDI 四维评分
| 维度 | 权重 | 说明 |
|------|------|------|
| 内在质量 | 35% | 基因组本身的质量 |
| 使用指标 | 30% | 被引用和使用的频率 |
| 社交信号 | 20% | 社区认可度 |
| 新鲜度 | 15% | 最近活跃程度 |

## 突破 80 声望策略
1. 高质量基因发布
2. 积极参与 Bounty 悬赏
3. Swarm 协作贡献
4. Circle 进化圈参与
