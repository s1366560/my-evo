# Troubleshooting Deadlocks in Multi-Agent Resource Allocation with Semaphores

## Problem Statement

Deadlocks in multi-agent systems occur when two or more agents hold locks on resources and wait for each other to release their locks, creating a circular dependency. This capsule provides a comprehensive debugging and resolution framework for semaphore-based resource allocation deadlocks.

## Core Concepts

### What is a Deadlock?

A deadlock occurs when a set of processes are each waiting for some action from another process in the set, and none can proceed. The four Coffman conditions must ALL be met:

1. **Mutual Exclusion**: Resource is held in non-sharable mode
2. **Hold and Wait**: Process holds resource while waiting for another
3. **No Preemption**: Resources cannot be forcibly taken
4. **Circular Wait**: Circular chain of processes each waiting for a resource held by the next

### Semaphore-Based Resource Allocation

Semaphores are synchronization primitives that use atomic operations to control access to shared resources:

```python
import threading
from typing import Dict, List, Set
from dataclasses import dataclass, field
from enum import Enum
import time
import random

class Semaphore:
    """Classic semaphore implementation with atomic wait/signal"""
    
    def __init__(self, value: int = 1):
        self._value = value
        self._wait_queue: List[threading.Thread] = []
        self._lock = threading.Lock()
    
    def acquire(self, blocking: bool = True, timeout: float = -1) -> bool:
        """Acquire the semaphore"""
        with self._lock:
            if self._value > 0:
                self._value -= 1
                return True
            if not blocking:
                return False
        # Wait on queue (simplified)
        start = time.time()
        while True:
            with self._lock:
                if self._value > 0:
                    self._value -= 1
                    return True
            if timeout > 0 and (time.time() - start) >= timeout:
                return False
            time.sleep(0.001)
    
    def release(self):
        """Release the semaphore"""
        with self._lock:
            self._value += 1
    
    def __enter__(self):
        self.acquire()
        return self
    
    def __exit__(self, *args):
        self.release()
```

## Deadlock Detection Strategies

### 1. Resource Allocation Graph (RAG) Analysis

```python
from typing import Dict, List, Set, Tuple, Optional
from collections import defaultdict
import graphviz

class ResourceAllocationGraph:
    """
    RAG tracks the state of resource allocation and request edges.
    - Process -> Resource edge: "Process is waiting for Resource"
    - Resource -> Process edge: "Resource is allocated to Process"
    """
    
    def __init__(self):
        self.processes: Set[str] = set()
        self.resources: Set[str] = set()
        # Allocation: resource -> {processes}
        self.allocation: Dict[str, Set[str]] = defaultdict(set)
        # Request: process -> {resources}
        self.requests: Dict[str, Set[str]] = defaultdict(set)
    
    def request(self, process: str, resource: str):
        """Process requests a resource"""
        self.processes.add(process)
        self.resources.add(resource)
        self.requests[process].add(resource)
    
    def allocate(self, process: str, resource: str):
        """Resource is allocated to process"""
        self.requests[process].discard(resource)
        self.allocation[resource].add(process)
    
    def release(self, process: str, resource: str):
        """Process releases a resource"""
        self.allocation[resource].discard(process)
    
    def detect_cycle(self) -> Optional[List[str]]:
        """Detect if there's a cycle in the RAG (indicates deadlock)"""
        # Build adjacency for processes only
        adj = defaultdict(list)
        for proc, res_set in self.requests.items():
            for res in res_set:
                # Find which processes hold this resource
                for holder in self.allocation.get(res, []):
                    adj[proc].append(holder)
        
        # DFS cycle detection
        visited = set()
        rec_stack = set()
        path = []
        
        def dfs(node):
            visited.add(node)
            rec_stack.add(node)
            path.append(node)
            
            for neighbor in adj[node]:
                if neighbor not in visited:
                    if dfs(neighbor):
                        return True
                elif neighbor in rec_stack:
                    # Found cycle
                    idx = path.index(neighbor)
                    return path[idx:]
            
            path.pop()
            rec_stack.remove(node)
            return False
        
        for proc in self.processes:
            if proc not in visited:
                cycle = dfs(proc)
                if cycle:
                    return cycle
        return None
    
    def visualize(self, filename: str = "rag.dot"):
        """Generate RAG visualization"""
        dot = graphviz.Digraph()
        dot.attr(rankdir='LR')
        
        # Processes (rectangles)
        for p in self.processes:
            dot.node(p, shape='box', label=f"P:{p}")
        
        # Resources (circles with capacity)
        for r in self.resources:
            dot.node(r, shape='circle', label=f"R:{r}")
        
        # Request edges (dashed)
        for p, res_set in self.requests.items():
            for r in res_set:
                dot.edge(p, r, style='dashed', color='red')
        
        # Allocation edges (solid)
        for r, proc_set in self.allocation.items():
            for p in proc_set:
                dot.edge(r, p, style='solid', color='green')
        
        dot.render(filename, format='png', cleanup=True)
```

### 2. Banker's Algorithm (Prevention)

```python
from typing import Dict, List, Set, Optional
import numpy as np

class BankersAlgorithm:
    """
    Deadlock prevention using Banker's Algorithm.
    Requires knowledge of maximum resource needs upfront.
    """
    
    def __init__(self, available: List[int], maximum: List[List[int]]):
        """
        available: available instances of each resource type
        maximum: max demand of each process (n_processes x n_resource_types)
        """
        self.available = np.array(available)
        self.maximum = np.array(maximum)
        self.n_processes = len(maximum)
        self.n_resources = len(available)
        # Initially no allocation
        self.allocation = np.zeros((self.n_processes, self.n_resources), dtype=int)
        # Need = Maximum - Allocation
        self.need = self.maximum - self.allocation
    
    def is_safe_state(self, available: Optional[np.array] = None) -> Tuple[bool, List[int]]:
        """
        Check if system is in safe state.
        Returns (is_safe, safe_sequence)
        """
        work = available.copy() if available is not None else self.available.copy()
        finish = [False] * self.n_processes
        safe_sequence = []
        
        for _ in range(self.n_processes):
            found = False
            for i in range(self.n_processes):
                if not finish[i] and np.all(self.need[i] <= work):
                    # Can satisfy needs
                    work += self.allocation[i]
                    finish[i] = True
                    safe_sequence.append(i)
                    found = True
            
            if not found:
                return False, []
        
        return True, safe_sequence
    
    def request(self, process_id: int, request: List[int]) -> Tuple[bool, str]:
        """
        Request resources. Returns (granted, reason)
        """
        request = np.array(request)
        
        # Step 1: Request <= Need
        if not np.all(request <= self.need[process_id]):
            return False, "Request exceeds declared maximum"
        
        # Step 2: Request <= Available
        if not np.all(request <= self.available):
            return False, "Insufficient resources available"
        
        # Step 3: Pretend to allocate
        test_available = self.available - request
        test_allocation = self.allocation.copy()
        test_allocation[process_id] += request
        test_need = self.need.copy()
        test_need[process_id] -= request
        
        # Step 4: Check safety
        is_safe, _ = self.is_safe_state(test_available)
        if is_safe:
            self.available = test_available
            self.allocation = test_allocation
            self.need = test_need
            return True, "Request granted"
        else:
            return False, "Request would lead to unsafe state"
```

## Practical Debugging Framework

### 1. Deadlock Detector Service

```python
import asyncio
import logging
from typing import Dict, List, Set, Optional
from dataclasses import dataclass, field
from enum import Enum
from collections import deque
import threading
import time

class DeadlockDetector:
    """
    Periodic deadlock detection using wait-for graph analysis.
    Suitable for multi-agent systems with semaphore-based resource allocation.
    """
    
    def __init__(self, check_interval: float = 5.0):
        self.check_interval = check_interval
        self.running = False
        self.logger = logging.getLogger(__name__)
        
        # Resource state
        self.resources: Dict[str, 'Resource'] = {}
        self.processes: Dict[str, 'Process'] = {}
        self._lock = threading.RLock()
        
        # Detection callbacks
        self._on_deadlock: Optional[callable] = None
    
    def register_resource(self, resource_id: str, capacity: int = 1):
        with self._lock:
            self.resources[resource_id] = Resource(resource_id, capacity)
    
    def register_process(self, process_id: str):
        with self._lock:
            self.processes[process_id] = Process(process_id)
    
    def on_deadlock(self, callback: callable):
        """Set callback to be called when deadlock detected"""
        self._on_deadlock = callback
    
    def record_wait(self, process_id: str, resource_id: str):
        """Record that process is waiting for resource"""
        with self._lock:
            if process_id in self.processes and resource_id in self.resources:
                self.processes[process_id].waiting_for.add(resource_id)
                self.resources[resource_id].waiting_processes.add(process_id)
    
    def record_release(self, process_id: str, resource_id: str):
        """Record that process released resource"""
        with self._lock:
            if process_id in self.processes:
                self.processes[process_id].waiting_for.discard(resource_id)
            if resource_id in self.resources:
                self.resources[resource_id].waiting_processes.discard(process_id)
    
    def detect(self) -> Optional[List[str]]:
        """
        Detect deadlock using wait-for graph DFS.
        Returns cycle path if deadlock found, None otherwise.
        """
        with self._lock:
            # Build wait-for graph: process -> {processes it waits for}
            wait_for = defaultdict(set)
            for proc_id, proc in self.processes.items():
                for res_id in proc.waiting_for:
                    for holder_proc_id, holder_proc in self.processes.items():
                        if res_id in holder_proc.held_resources:
                            wait_for[proc_id].add(holder_proc_id)
            
            # Find cycle using DFS
            visited = set()
            rec_stack = set()
            path = []
            
            def dfs(node: str) -> Optional[List[str]]:
                visited.add(node)
                rec_stack.add(node)
                path.append(node)
                
                for neighbor in wait_for.get(node, []):
                    if neighbor not in visited:
                        result = dfs(neighbor)
                        if result:
                            return result
                    elif neighbor in rec_stack:
                        idx = path.index(neighbor)
                        return path[idx:] + [neighbor]
                
                path.pop()
                rec_stack.remove(node)
                return None
            
            for proc_id in self.processes:
                if proc_id not in visited:
                    cycle = dfs(proc_id)
                    if cycle:
                        return cycle
            return None
    
    async def start(self):
        """Start periodic deadlock detection"""
        self.running = True
        while self.running:
            await asyncio.sleep(self.check_interval)
            cycle = self.detect()
            if cycle:
                self.logger.error(f"Deadlock detected: {' -> '.join(cycle)}")
                if self._on_deadlock:
                    await self._on_deadlock(cycle)

@dataclass
class Resource:
    resource_id: str
    capacity: int
    waiting_processes: Set[str] = field(default_factory=set)

@dataclass
class Process:
    process_id: str
    held_resources: Set[str] = field(default_factory=set)
    waiting_for: Set[str] = field(default_factory=set)
```

### 2. Agent-Safe Semaphore Wrapper

```python
import asyncio
from typing import Optional, Set
import logging

class DeadlockFreeSemaphore:
    """
    Semaphore wrapper that implements timeout-based acquisition
    to prevent indefinite blocking and potential deadlocks.
    """
    
    def __init__(self, value: int, name: str = "unnamed", timeout: float = 10.0):
        self._semaphore = asyncio.Semaphore(value)
        self.name = name
        self.timeout = timeout
        self.logger = logging.getLogger(f"sem.{name}")
        self._holders: Set[int] = set()
    
    async def acquire(self, agent_id: str) -> bool:
        """Acquire with timeout and logging"""
        try:
            await asyncio.wait_for(
                self._semaphore.acquire(),
                timeout=self.timeout
            )
            self._holders.add(hash(agent_id))
            self.logger.debug(f"Agent {agent_id} acquired {self.name}")
            return True
        except asyncio.TimeoutError:
            self.logger.warning(
                f"Agent {agent_id} timed out waiting for {self.name}. "
                f"Potential deadlock - investigating..."
            )
            # Trigger deadlock detection
            self._on_timeout(agent_id)
            return False
    
    def release(self, agent_id: str):
        """Release semaphore"""
        self._semaphore.release()
        self._holders.discard(hash(agent_id))
        self.logger.debug(f"Agent {agent_id} released {self.name}")
    
    def _on_timeout(self, agent_id: str):
        """Hook for timeout handling"""
        pass
    
    async def __aenter__(self):
        return self
    
    async def __aexit__(self, *args):
        pass

class HierarchicalLockManager:
    """
    Prevents deadlocks by enforcing lock ordering.
    Agents must acquire locks in a global consistent order.
    """
    
    def __init__(self):
        self._locks: Dict[str, asyncio.Lock] = {}
        self._acquired: Set[str] = set()
        self._order: Dict[str, int] = {}  # Global ordering
        self._lock = asyncio.Lock()
    
    def set_lock_order(self, lock_names: List[str]):
        """Define the global lock ordering"""
        self._order = {name: idx for idx, name in enumerate(lock_names)}
    
    async def acquire(self, name: str, agent_id: str):
        """Acquire lock in global order"""
        if name not in self._order:
            raise ValueError(f"Unknown lock: {name}")
        
        # Acquire all locks with lower order first
        needed_order = self._order[name]
        with self._lock:
            to_acquire = sorted(
                [l for l in self._order if l not in self._acquired and self._order[l] < needed_order],
                key=lambda x: self._order[x]
            )
        
        for lock_name in to_acquire:
            await self._locks[lock_name].acquire()
            self._acquired.add(lock_name)
        
        await self._locks[name].acquire()
        self._acquired.add(name)
    
    async def release(self, name: str):
        """Release all locks held by agent in reverse order"""
        with self._lock:
            to_release = sorted(
                [l for l in self._acquired if l == name or self._order[l] >= self._order.get(name, -1)],
                key=lambda x: -self._order[x]
            )
        
        for lock_name in to_release:
            self._locks[lock_name].release()
            self._acquired.discard(lock_name)
```

## Common Deadlock Patterns and Fixes

### Pattern 1: Nested Lock Acquisition

**Problem:**
```python
# Agent A                    # Agent B
lock1.acquire()             lock2.acquire()
# ...                       # ...
lock2.acquire()             lock1.acquire()  # DEADLOCK
```

**Fix: Consistent Lock Ordering**
```python
# Both agents must acquire in order: lock1 -> lock2
async def agent_work(agent_id):
    # Always acquire in consistent order
    ordered_locks = sorted([lock1, lock2], key=lambda x: x.name)
    for lock in ordered_locks:
        await lock.acquire(agent_id)
    try:
        # work
    finally:
        for lock in reversed(ordered_locks):
            lock.release(agent_id)
```

### Pattern 2: Unbounded Resource Waiting

**Problem:** Agent waits indefinitely for resources with no timeout.

**Fix: Timeout-Based Acquisition**
```python
async def request_resource(agent_id, resource, timeout=5.0):
    try:
        await asyncio.wait_for(resource.acquire(), timeout=timeout)
        return True
    except asyncio.TimeoutError:
        logger.error(f"Agent {agent_id} timed out waiting for {resource}")
        return False
```

### Pattern 3: Circular Wait

**Problem:** A→B→C→A wait chain.

**Fix: Single Resource Policy / Resource Hierarchy**
```python
class ResourceHierarchy:
    """Enforce total ordering on resources to break circular wait"""
    def __init__(self):
        self._resources: Dict[str, asyncio.Lock] = {}
        self._order_counter = 0
    
    async def acquire_all(self, agent_id: str, resource_names: List[str]):
        # Sort by hierarchy - all agents must agree on order
        ordered = sorted(resource_names)
        for name in ordered:
            await self._resources[name].acquire()
            self.logger.debug(f"Agent {agent_id} acquired {name}")
```

## Multi-Agent Resource Allocation Best Practices

1. **Always use timeouts** on all acquisition operations
2. **Implement lock ordering** for multiple resource scenarios
3. **Use Banker's Algorithm** when maximum resource needs are known
4. **Monitor for cycles** in the wait-for graph
5. **Implement deadlock recovery** (process termination, resource preemption)
6. **Log all acquisition/release** operations for forensic analysis
7. **Pre-allocate resources** when possible to reduce contention
8. **Use resource pooling** instead of per-agent resource allocation

## Summary

Deadlock debugging in semaphore-based multi-agent systems requires:

1. **Detection**: RAG analysis, wait-for graph DFS
2. **Prevention**: Banker's Algorithm, lock ordering, single resource policy
3. **Avoidance**: Timeout-based acquisition, deadlock-free data structures
4. **Recovery**: Process termination, resource preemption, rollback

The combination of proactive prevention (consistent lock ordering, hierarchical acquisition) and reactive detection (periodic wait-for graph analysis) provides the most robust approach to handling deadlocks in production multi-agent systems.
