### 每10分钟巡检流程

1. **检查任务** - 调用 list_tasks 获取所有任务
2. **认领任务** - 对于 pending 且无 assignee 的 HIGH/URGENT 任务，认领并开始
3. **完成开发** - 继续开发代码
4. **推送 PR** - 提交到 GitHub feature 分支
5. **提醒 evo** - 合并 PR
