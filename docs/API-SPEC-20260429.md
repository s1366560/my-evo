# My Evo API Specification

**Version:** 1.0.0 | **Format:** OpenAPI 3.0.3 | **Base URL:** `http://localhost:3001`

---

## Authentication

All protected endpoints require JWT Bearer Token:

```
Authorization: Bearer <jwt_token>
```

### Response Formats

**Success:**
```json
{ "data": {...}, "token": "eyJ..." }
```

**Error:**
```json
{ "error": "message", "code": "CODE" }
```

---

## HTTP Status Codes

| Code | Description |
|------|-------------|
| 200 | Success |
| 201 | Created |
| 204 | No Content |
| 400 | Validation Error |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
| 409 | Conflict |
| 500 | Internal Error |

## Core Schemas

### User
```yaml
User:
  id: string (uuid)
  email: string (email)
  name: string
  avatar: string
  role: string (user|admin)
  level: integer
  reputation: number
  credits: number
  claimed: boolean
  claimCode: string
  createdAt: datetime
  updatedAt: datetime
```

### Node
```yaml
Node:
  id: string (uuid)
  name: string
  description: string
  type: string (default: concept)
  category: string
  metadata: object
  position: object {x, y}
  level: integer
  reputation: number
  viewCount: integer
  ownerId: string
  createdAt: datetime
  updatedAt: datetime
```

### Edge
```yaml
Edge:
  id: string (uuid)
  sourceId: string
  targetId: string
  type: string (default: related)
  weight: number (default: 1.0)
  description: string
  createdAt: datetime
```

### GraphData
```yaml
GraphData:
  nodes: Node[]
  edges: Edge[]
```

## Endpoints Summary

### Health Check
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/health` | No | Server health |

### Authentication (prefix: `/api/v1/auth`)
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| POST | `/register` | No | Register user |
| POST | `/login` | No | Login |
| POST | `/logout` | Yes | Logout |
| GET | `/me` | Yes | Current user |

### Map - Nodes (prefix: `/api/v1/map/nodes`)
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/` | No | List nodes |
| POST | `/` | Yes | Create node |
| GET | `/:id` | No | Get node |
| PUT | `/:id` | Yes | Update node |
| DELETE | `/:id` | Yes | Delete node |

### Map - Edges (prefix: `/api/v1/map/edges`)
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/` | No | List edges |
| POST | `/` | Yes | Create edge |
| DELETE | `/:id` | Yes | Delete edge |

### Graph (prefix: `/api/v1/graph`)
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| POST | `/layout` | No | Compute layout |
| POST | `/layout/force` | No | Force-directed |
| POST | `/layout/grid` | No | Grid community |
| POST | `/layout/radial` | No | Radial hierarchy |
| POST | `/validate` | No | Validate graph |

### Dashboard (prefix: `/api/v2/dashboard`)
| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/stats` | Yes | User statistics |

| GET | `/credits` | Yes | Credit balance |
| GET | `/reputation` | Yes | Reputation metrics |

## Detailed Endpoints

### POST /api/v1/auth/register

**Request:**
```json
{
  "email": "user@example.com",
  "password": "password123",
  "name": "John Doe"
}
```

**Response (201):**
```json
{
  "data": {
    "id": "uuid",
    "email": "user@example.com",
    "name": "John Doe",
    "role": "user",
    "level": 1,
    "reputation": 0,
    "credits": 0,
    "createdAt": "2026-04-29T00:00:00Z"
  },
  "token": "eyJhbGciOiJIUzI1NiIs..."
}
```

### POST /api/v1/auth/login

**Request:**
```json
{
  "email": "user@example.com",
  "password": "password123"
}
```

**Response (200):**
```json
{
  "data": { ... },
  "token": "eyJ..."
}
```

### GET /api/v1/map/nodes

**Query Parameters:**
- `type` (string): Filter by node type
- `category` (string): Filter by category
- `limit` (integer, default: 50)
- `offset` (integer, default: 0)

**Response (200):**
```json
{
  "data": [Node, ...],
  "total": 100,
  "limit": 50,
  "offset": 0
}
```

### POST /api/v1/map/nodes

**Request:**
```json
{
  "name": "Machine Learning",
  "description": "A field of AI",
  "type": "concept",
  "category": "technology",
  "metadata": { "tags": ["AI", "ML"] }
}
```

**Response (201):**
```json
{
  "data": Node
}
```

### POST /api/v1/graph/layout

**Request:**
```json
{
  "nodes": [{ "id": "1", "name": "A" }],
  "edges": [{ "sourceId": "1", "targetId": "2" }],
  "algorithm": "force",
  "options": { "iterations": 300 }
}
```

**Response (200):**
```json
{
  "nodes": [{ "id": "1", "position": { "x": 0, "y": 0 } }],
  "layoutTime": 45
}
```
