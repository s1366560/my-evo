## 3. 数据流图对比

### 3.1 架构文档定义

```
[浏览器] ←→ [Next.js SSR/CSR] ←→ [Nginx 反向代理] ←→ [Fastify API]
                │                                       │
         [React 组件树]                          [Prisma ORM]
         [Zustand 状态]                          [PostgreSQL]
         [React Query]                           [Redis 缓存]
```

### 3.2 实际实现

```
[浏览器] ←→ [Next.js 14 App Router] ←→ [Express API (localhost:3001)]
                │                              │
         [React + Zustand]            [Prisma ORM]
         [Canvas 2D]                  [SQLite]
```

**差异说明**:
- 使用 Express 替代 Fastify（功能等价）
- 使用 SQLite 替代 PostgreSQL（开发环境）
- 无 Nginx 层（开发环境直连）
- 无 Redis 层（当前版本未实现缓存）
