## 3. 发现的问题 — Swarm 状态机

### 问题描述

Phase 3 Swarm 实现缺少 3 个关键状态转换，违反架构文档 (Chapter 5/33) 定义的状态机规范：

| # | 问题 | 架构期望 | 实际行为 |
|---|------|---------|---------|
| 1 | `submitDecomposition` 未更新 swarm 状态 | `idle` → `decomposition` | 停留在 `idle` |
| 2 | `rejectProposal` 未重置 swarm 状态 | → `idle` | 停留在 `decomposition` |
| 3 | `submitAggregatedResult` 跳过 `aggregating` 状态 | `solving` → `aggregating` → `completed` | 直接 → `completed` |

### 修复方案

**src/swarm/engine.ts**

```typescript
// 1. submitDecomposition →  transition to 'decomposition'
export function submitDecomposition(proposal) {
  proposals.set(proposal.swarm_id, p);
  updateSwarmState(proposal.swarm_id, 'decomposition'); // 新增
  return p;
}

// 2. rejectProposal → reset to 'idle'
export function rejectProposal(swarmId) {
  p.status = 'rejected';
  updateSwarmState(swarmId, 'idle'); // 新增
  return p;
}

// 3. submitAggregatedResult → through 'aggregating' → 'completed'
export function submitAggregatedResult(result) {
  results.set(result.swarm_id, r);
  updateSwarmState(result.swarm_id, 'aggregating'); // 新增
  updateSwarmState(result.swarm_id, 'completed');
  return r;
}

// 4. updateSubtaskState → auto transition to aggregating
export function updateSubtaskState(subtaskId, state, result?) {
  // ... existing logic ...
  if (state === 'completed') {
    tryTransitionToAggregating(subtask.swarm_id); // 新增
  }
  return subtask;
}

// 5. New helper
export function tryTransitionToAggregating(swarmId) {
  const swarm = swarms.get(swarmId);
  if (!swarm || swarm.state !== 'solving') return undefined;
  if (!areAllSubtasksComplete(swarmId)) return undefined;
  return updateSwarmState(swarmId, 'aggregating');
}
```
