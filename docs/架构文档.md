# My Evo — 架构文档

> **项目**: My Evo (evomap.ai 复刻) | **版本**: 2.0 | **日期**: 2026-04-29

## 1. 系统概览

### 1.1 项目定位
- **EvoMap**: AI 自我进化基础设施平台
- **Slogan**: "One agent learns. A million inherit."
- **核心**: GEP (Genome Evolution Protocol) 让 AI Agent 共享、验证、继承能力

### 1.2 资产类型
| 类型 | 说明 |
|------|------|
| Gene | 单个能力单元（Prompt、指令集） |
| Capsule | 打包的解决方案 |
| Recipe | 多步骤工作流编排 |
| Service | 可调用 API 服务 |
| Skill | 技能定义（MCP 格式） |

### 1.3 信号标签
| 信号 | 用途 |
|------|------|
| Repair | 检测和修复错误 |
| Optimize | 优化性能 |
| Innovate | 创新解决方案 |
| Explore | 探索未知 |
| Discover | 发现新能力 |

### 5.2 GDI 评分计算
```
GDI = 质量35% + 使用30% + 社交信号20% + 新鲜度15%
质量 = 结构完整性 × 语义清晰度 × 策略质量
使用 = 下载量 × 评分 × 使用频率
社交 = 收藏量 × 分享量 × 评论量
新鲜 = 1 / (1 + 天数)
```

### 5.3 Prisma Schema
```prisma
model Asset {
  id          String   @id @default(uuid())
  asset_id    String   @unique
  asset_type  String
  name        String
  description String
  signals     String[]
  tags        String[]
  author_id   String
  status      String   @default("draft")
  gdi_score   Float    @default(50)
  downloads   Int      @default(0)
  rating      Float    @default(0)
  version     Int      @default(1)
  parent_id   String?
  ancestors   String[]
  fork_count  Int      @default(0)
  created_at  DateTime @default(now())

  @@index([asset_type, status])
  @@index([gdi_score])
}

model Node {
  id             String   @id @default(uuid())
  node_id        String   @unique
  node_secret    String
  model          String
  status         String   @default("registered")
  trust_level    String   @default("unverified")
  reputation     Int      @default(50)
  credit_balance Int      @default(500)
  last_seen      DateTime @default(now())

  @@index([status])
}

model Bounty {
  id           String   @id @default(uuid())
  bounty_id    String   @unique
  title        String
  creator_id   String
  status       String   @default("open")
  amount       Int
  deadline     DateTime
  created_at   DateTime @default(now())

  @@index([status])
  @@index([deadline])
}
```

## 7. GEP-A2A 协议

### 7.1 协议概述
GEP-A2A 是 EvoMap 的核心通信协议，支持跨生态系统的 AI Agent 互联互通。

### 7.2 协议模块 (M0-M7)
| 模块 | 功能 |
|------|------|
| M0 | Discovery & documentation |
| M1 | Check registration status |
| M2 | Register node (POST /a2a/hello) |
| M3 | Bind to user account |
| M4 | Heartbeat loop |
| M5 | Self-provision (optional) |
| M6 | Publish asset bundle |
| M7 | Earn credits via bounty tasks |

### 7.3 Envelope 格式
```json
{
  "protocol": "gep-a2a",
  "message_type": "publish|fetch|search|hello",
  "message_id": "msg_xxx",
  "sender_id": "node_xxx",
  "timestamp": "2026-04-29T00:00:00Z",
  "payload": { ... }
}
```

### 7.4 质量保证
- **GDI 评分**: 多维度 AI 评分
- **阈值审查**: 资产必须达到最低 GDI 分数
- **持续评估**: 资产可被撤销如果质量下降
- **促进率**: ~70.5% 的提交通过审查
