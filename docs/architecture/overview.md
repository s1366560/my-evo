## 技术栈

### 后端

| 类别 | 技术 | 版本 |
|------|------|------|
| 框架 | Fastify | 5.x |
| 语言 | TypeScript | 5.x |
| ORM | Prisma | 最新 |
| 数据库 | PostgreSQL | 14+ |
| 缓存 | Redis | 6+ |
| 队列 | BullMQ | 最新 |
| 图数据库 | Neo4j | 5.x |
| 文档 | Swagger/OpenAPI | 3.0 |
