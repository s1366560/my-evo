# My Evo 核心交互流程图

**版本**: 1.0 | **日期**: 2026-04-29 | **类型**: 交互流程文档

---

## 1. 用户旅程总览

```
访问落地页 → 注册登录 → 首次引导 → 探索市场 → 参与贡献
     ↓            ↓           ↓           ↓           ↓
  浏览资产 ← 购买/发布 ← 赚取积分 ← 提升声誉 ← 社区贡献
```

---

## 2. 注册与引导流程

```
[落地页] → [注册表单] → [邮箱验证/社交登录] → [个人资料] → [角色选择]
                                                       ↓
                     ┌────────────┬────────────┬──────┴──────┐
                     ↓            ↓            ↓              ↓
                 [发布者]    [消费者]    [开发者]      [进入仪表盘]
```

---

## 3. 资产发布流程

```
[仪表盘] → [发布类型选择: Gene/Capsule/Recipe]
                    ↓
           ┌────────┴────────┐
           ↓                 ↓
        [Gene表单]      [Capsule表单]   → 3.预览验证 → 4.定价策略 → 5.提交审核
           ↓                 ↓                 ↓
        基本信息         Gene组合          发布成功
           ↓                 ↓
        代码内容         执行路径
```

## 4. 资产购买流程

```
[市场] → [资产列表] → [详情页] → [购买/加入购物车] → [余额检查]
                                              ↓
                         ┌────────────────────┴────────────────────┐
                         ↓                                         ↓
                   [余额充足]                                  [余额不足]
                         ↓                                         ↓
                   [确认购买]                                  [引导充值]
                         ↓                                         ↓
                   [积分扣除+下载]                              [充值页面]
                         ↓
                   [使用/评价]
```

---

## 5. 赏金任务流程

### 发布者:
```
[仪表盘] → [创建赏金] → [填写详情] → [发布] → [认领等待] → [方案验收] → [赏金发放]
```

### 解决者:
```
[赏金大厅] → [浏览筛选] → [认领任务] → [解决问题] → [提交方案] → [验收通过获得赏金]
```
