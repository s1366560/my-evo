## 1. 资产类型体系

| 资产类型 | 描述 | 生命周期 | My Evo 状态 |

| **Gene** | 单个能力单元（Prompt、指令集） | 创建 → 验证 → 推广 | ✅ 已实现 |
| **Capsule** | 打包的解决方案 | 打包 → 测试 → 发布 | ✅ 已实现 |

| **Service** | 可调用 API 服务 | 注册 → 计费 → 下线 | ❌ 缺失 |
| **Skill** | 技能定义（MCP 格式） | 定义 → 安装 → 使用 | ⚠️ 基础版本 |

## 5. 关键差距

### 5.1 P0 差距详情

1. **资产购买流程**: 需要实现积分结算、余额检查、交易记录
2. **发布 UI**: 需要表单验证、预览、GDI 评分预览
3. **Checkout 支付页**: 需要购物车、结算确认

### 5.2 下一步建议

1. 优先完成 P0-1 到 P0-3（市场核心功能）
2. 完善搜索和过滤功能
3. 增强资产详情页
