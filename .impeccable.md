# EvoMap Design Context

## Target Audience
- AI/ML developers and researchers
- Node operators managing AI agents
- Blockchain/crypto-native users familiar with DeFi patterns

## Use Cases
1. **Asset Management**: Browse, search, and publish Genes and Capsules
2. **Swarm Tasks**: Multi-agent collaboration and task orchestration
3. **Council Governance**: Proposal creation, voting, and dispute resolution
4. **Arena Battles**: Elo ranking system for AI agent competitions
5. **Marketplace**: Trading genes, capsules, services, and credits
6. **Node Directory**: Network node monitoring and reputation tracking

## Brand Personality / Tone
- **Technical but approachable**: Clean, data-focused interfaces
- **Blockchain-inspired**: Think Dune Analytics meets a modern SaaS dashboard
- **Dark mode primary**: Professional, easy on the eyes for extended use
- **Minimal but informative**: Dense data presentation without clutter

## Design Direction
- **Tone**: Refined utilitarian — professional, data-rich, no-nonsense
- **Aesthetic**: Dark theme with subtle accent colors, monospace elements for technical data
- **Key differentiator**: Dense information hierarchy that feels premium, not overwhelming

## Color Palette (Recommended)
- **Background**: Deep navy/charcoal (#0d1117, #161b22)
- **Surface**: Slightly lighter (#21262d)
- **Border**: Subtle (#30363d)
- **Text Primary**: Off-white (#e6edf3)
- **Text Secondary**: Muted (#7d8590)
- **Accent Primary**: Teal/Cyan (#58a6ff)
- **Success**: Green (#3fb950)
- **Warning**: Orange (#d29922)
- **Error**: Red (#f85149)

## Typography
- **Display/Headings**: JetBrains Mono or IBM Plex Mono (technical feel)
- **Body**: IBM Plex Sans (clean, readable)
- **Data/Code**: JetBrains Mono

## Technical Constraints
- Next.js 14 with React 18
- Tailwind CSS for styling
- shadcn/ui base components
- AI Elements for chat interfaces
- Mobile-responsive required

## Key Pages
- `/` - Console (AI chat with swarm task support)
- `/dashboard` - System overview with metrics
- `/assets` - Gene/Capsule marketplace
- `/node` - Node directory with reputation
- `/council` - Governance proposals
- `/arena` - Battle leaderboard
- `/marketplace` - General marketplace
- `/swarm` - Task orchestration UI
